Construct a class of airport routes that contains:

addRoute(start, destination): adds a possible route from start airport to destination airport
printRoutes(start, destination): prints all possible routes between two destinations

Example:
addRoute("A", "B")
addRoute("A", "C")
addRoute("B", "D")
addRoute("C", "D")

printRoutes("A", "D") should return [["A", "B", "D"], ["A", "C", "D"]]

ASSUMPTIONS:
- Every Airport will be represented as a single character string
- Will always be alphabets, and "A" != "a"
- If no valid routes, print your empty array
